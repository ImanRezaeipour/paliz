- ایجاد محلهای پرداخت

برای ایجاد یک محل جدید کافیست برروی گزینه کلیک کرده یا از کلید ترکیبی (Alt+N) استفاده نمایید. سپس نام عنوان را تعیین کنید. كد ارائه شده بصورت خودكارتوسط سيستم ايجاد مي شود و «·» Â قابل تغییر„ی باشد. پس ازورود ⁄‰Ê«‰ منطقه و«Õی«‰« ‘„«—Â ¬‰ „Ê””Â و کلیک بر روی آیکون   آن محل ایجاد می گردد.

 

- اصلاح / حذف محلهای جغرافیایی

اصلاح: برای اصلاح یک محل کافیست برروی گزینه کلیک کرده یا از کلید ترکیبی (Alt+M) استفاده نمایید. پس از اعمال تغییرات مورد نظر گزینه را کلیک کنید.

حذف: برای حذف یک محل کافیست برروی گزینه   کلیک کرده یا از کلید ترکیبی (Alt+D) استفاده نمایید، برای حذف قطعی گزینه را کلیک کنید. البته توجه داشته باشید که برای حذف یک مجموعه ابتدا باید زیرمجموعه هایش حذف شوند.
