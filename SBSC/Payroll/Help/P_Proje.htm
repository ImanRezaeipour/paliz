صدور سند مالی

 

این فرم جهت معرفی پروژه های موجود در یک سازمان مورد استفاده قرار می گیرد که با انتخاب گزینه مذکور از منوی اطلاعات پایه گشوده می شود. اگر سازمانی از سیستم حضور و غیاب پیوست نیز استفاده کند، به دلیل ارتباط بین دیتابیس حضور و غیاب و حقوق ، اطلاعات فرم پروژه ها در دو سیستم مرتبط و مشترک خواهد بود. در ضمن امکان تخصیص مزایا به تفکیک پروژه ها در فرم  صدور سند مالی نیز وجود دارد.

 

             

               شکل 1: فرم معرفی پروژه ها

 

 ورود اطلاعات

با کلیک برروی گزینه یا استفاده ازکلید ترکیبی(Alt+N) امکان ورود اطلاعات فراهم می شود. پس از ورود اطلاعات مربوطه بر روی آیکون کلیک کنید. نام مدیر پروژه  با تایپ علامت%یا قسمتی از نام وی و فشردن کلیدEnter قابل تعیین است( البته این نام درصورت استفاده از سیستم حضور و غیاب برای تایید مدیر پروژه استفاده می شود).

 

 اصلاح / حذف اطلاعات

اصلاح: برای اصلاح اطلاعات برروی گزینه کلیک کرده یا از کلید ترکیبی(Alt+M) استفاده نمایید. پس ازاعمال تغییرات مورد نظر گزینه را کلیک کنید.

حذف: برای حذف پروژه روی گزینه کلیک یا ازکلیدترکیبی(Alt+D) استفاده نمایید، برای حذف قطعی گزینه راکلیک یا ازکلیدترکیبی(Ctrl+D) استفاده کنید.

 

در ضمن امکان جستجو و گزینش نیزدراین فرم وجود دارد.
